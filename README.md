# tasks-api
Prueba técnica de Backend - Sistema de gestión de tareas

API REST desarrollada en Node.js con Express para la gestión de tareas.
Permite a los usuarios crear, consultar, editar y eliminar tareas,identificando 
al usuario que realiza las solicitudes mediante autenticación con JWT.

El proyecto tiene una arquitectura de MVC y por el siguiente sistema estándar de carpetas:
src/
├── controllers/ # Lógica de los endpoints
├── routes/ # Definición de rutas
├── middlewares/ # Middlewares de autenticación JWT
├── models/ # Modelos de datos
├── config/ # Configuración de base de datos y entorno
└── app.js # Inicialización de la aplicación


Stack Utilizado:
- Node.js
- Express.js
- Javascript
- MySQL
- JWT para autenticación
